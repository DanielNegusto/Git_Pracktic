# Проект по обработке данных

## Описание проекта

Этот проект предназначен для обработки и маскирования номеров карт и счётов, а также для фильтрации и сортировки списка операций по дате и статусу.

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-repo/project-name.git
    ```

2. Перейдите в директорию проекта:
    ```bash
    cd project-name
    ```

3. Установите необходимые зависимости:
    ```bash
    # для первичной установки
     poetry install
    # для обновления
    poetry update
    ```

## Модуль decorators

### log

Декоратор `log` используется для автоматического логирования начала и конца выполнения функции, а также ее результатов или возникших ошибок.

### При успешном выполнении функции, в файл mylog.txt будет записано:
```
"function_name" ok
```
### При возникновении ошибки:
```
"function_name" error:  Inputs: (a, b), {}
```
### Если filename не указан, логи будут выводиться в консоль.

# Финансовое приложение

Это приложение предназначено для обработки и анализа финансовых транзакций. Включает функции для чтения транзакций из JSON-файлов, конвертации валют и логирования выполнения функций.

## Структура проекта

- `src/`
  - `external_api.py`: Модуль для работы с внешним API для конвертации валют.
  - `utils.py`: Модуль для утилитарных функций, включая чтение транзакций.
  - `data/`: Директория для хранения JSON-файлов с данными о транзакциях.
- `tests/`: Директория для хранения тестов.

## Функции

### Чтение транзакций из JSON-файла

Функция `read_transactions` читает JSON-файл с данными о финансовых транзакциях и возвращает список словарей.

```python
from src.utils import read_transactions
transactions = read_transactions("data/operations.json")
```
### Параметры:

file_path (str): Путь до JSON-файла.
### Возвращает:

List[Dict[str, Any]]: Список словарей с данными о транзакциях. Если файл пустой, содержит не список или не найден, возвращается пустой список.

## Конвертация валют
Функция convert_to_rub конвертирует сумму из заданной валюты в рубли. Использует внешнее API для получения текущего курса валют.

### Получение информации по API
 Если транзакция была в USD или EUR , происходит обращение к внешнему API для получения текущего курса валют и конвертации суммы операции в рубли.
 Для конвертации валюты воспользуйтесь Exchange Rates Data API: https://apilayer.com/exchangerates_data-api 
Поместите API ключ в файл .env

### Вызов функции
```python
from src.external_api import convert_to_rub
amount_in_rub = convert_to_rub(100.0, "USD")
```
### Параметры:

amount (float): Сумма в исходной валюте.
currency (str): Код исходной валюты (например, 'USD', 'EUR').
### Возвращает:

float: Сумма в рублях.
